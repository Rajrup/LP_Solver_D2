Get Gurob Academic Licence: https://www.gurobi.com/academia/academic-program-and-licenses/

Add licence path in bashrc file:
`export GRB_LICENSE_FILE="/home/rajrup/license/gurobi/gurobi.lic"`

## Run LP
`python main_d2_lp.py`
